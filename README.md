# Анализ поведения покупателей (Shopping Behavior)

Проект выполнен в рамках курса DataSc101.

## Что сделано:
1. **Python (Pandas/Seaborn):** Проведен EDA (исследовательский анализ данных), обработаны пропуски и дубликаты, построены графики распределения возраста, категорий и корреляций.
2. **Yandex DataLens:** Создан интерактивный дашборд с аналитикой продаж и фильтрацией по сезонам.

## Файлы:
- `shopping_behavior.csv`: исходный датасет.
- `shopping_analysis_ru.ipynb`: ноутбук с кодом анализа.
